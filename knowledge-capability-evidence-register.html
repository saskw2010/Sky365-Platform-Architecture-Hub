Validated interpretation

  • The Controller is the strongest metadata center for screens, CRUD, lookups, validation and tool discovery.
  • It is not yet the sole source for agents, prompts, policies, RAG, dashboards, reports or multimodal pipelines.
  • Legacy Code On Time Controllers contain richer UI/action metadata than the simplest current conversion preserves.

Promotion roadmap

  1. Adopt scoped Controller identity and a versioned Controller Intermediate Representation.
  2. Create golden conversion and generated-screen parity evidence.
  3. Define policy/provenance inheritance across UI, CRUD, API, reports, tools and agents.
  4. Promote semantic, prompt/context and RAG only after contracts and evaluation gates exist.
