Extension model and dependencies

Current extension is primarily compile/startup-time DI, interfaces, components and adapters. It must not be confused with a governed Plugin or Extension Engine. Cross-engine dependencies converge on stable Controller identity, authorization, persistence, audit and metadata provenance.

Roadmap

Promotion requires purpose, owner, runtime boundary, inputs/outputs, metadata, dependencies, extension points, failure semantics and verification evidence. See the Maturity Matrix and Planned Engine Roadmap.
