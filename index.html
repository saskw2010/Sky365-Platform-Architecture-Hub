Validated platform thesis

SKY365 is not an ERP application architecture. It is an MDAP whose authored Controller and schema metadata drives UI, CRUD, lookup, validation, workflow links, navigation, reporting and tool behavior. ERP and vertical solutions are applications built on or beside that platform.

The thesis is qualified: one Controller does not yet own every capability. Workflow, dashboard, report and AI metadata include linked specialized domains. See ADR-0001, ADR-0002, and ADR-0005.

Evidence policy

  • Verified: executable source and an identifiable runtime path exist at a pinned revision.
  • Partial: useful components exist, but the complete governed engine or end-to-end coverage is not demonstrated.
  • Prototype/documented: an experiment or architecture exists without production lifecycle evidence.
  • Not found: no supporting implementation was established in the inspected scope; this is not proof of global absence.

No production code is modified by this discovery mission.
